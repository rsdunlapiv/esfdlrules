QuestionnaireResponseParser:

This class holds methods to parse QuestionnaireResponse.

The following is the list of methods with their corresponding functionalities:

getModelingInfrastructureName - returns the name of the modelling infrastructure corresponding to QuestionnaireResponse.
 
getRootNode - returns the root element of the Questionnaire response XML file.

parsePropertiesAndValues - parses the Questionnaire response XML file and stores the properties and corresponding values in a HashMap propertiesAndValues.

getPropertiesAndValues - returns HashMap of properties and corresponding values.

Dependencies - 

Questionnaire Response XML file should be placed in xml folder.
